# 服装遮羞的失真——时尚的失败

对服装的考古揭示了其本质的多重性：它始于生存的铠甲（保暖与保护），演变为功能的延伸（携带工具），并很快成为一种原始的符号系统，用以沟通部落、地位与信仰。这种“遮羞”的功能，或许是文明最初的规训，将自然的身体纳入社会的符号秩序。法国思想家米歇尔·福柯在《规训与惩罚》中深刻地指出：“权力关系直接控制它，干预它，给它打上标记，训练它，折磨它，强迫它完成某些任务、表现某些仪式和发出某些信号。”服装正是这种微观权力的体现，它从文明伊始就参与了对身体的规划与管理。

德国社会学家格奥尔格·齐美尔在《时尚的哲学》中进一步揭示了这种符号系统的运作逻辑：“时尚只不过是我们众多寻求将社会一致化与个体差异化这两种对立的倾向，以一系列不断变化的妥协形式加以结合的生活领域之一。”时尚本质上是社会上层阶级用以区隔自身与下层的工具，而下层阶级的模仿行为则促使上层不断更新时尚符号，形成一个永恒的循环。这一理论精准地预言了前现代社会服装与身份的紧密对应关系——你的着装直接宣告了你在社会结构中的位置。


然而，正是这“沟通工具”的属性，将其引向了一个永恒的悖论：我们创造了服装以“完善”自身，它却反过来，成为规训身体、并放大我们欲望与荒诞的媒介。无论是日本AV中对校服的神圣化亵渎，还是维密秀将身体转化为梦幻商品，服装都扮演着同一个角色：它通过遮蔽来凸显、通过束缚来煽动，将活生生的肉体提炼为可供消费的欲望符号。在这里，“遮羞”的初衷被彻底扭曲，服装不再是道德的屏障，反而成了欲望的导流渠。让·鲍德里亚在《消费社会》中警告我们：“在消费的全套装备中，有一种比其他一切都更美丽、更珍贵、更光彩夺目的物品——它比负载了全部内涵的汽车还要负载了更沉重的内涵，这便是身体。”在消费时代，一切，包括身体和性，都被物化为符号和商品，服装正是这一过程的完美同谋。


在此，一种庸俗的辩护通常会响起：“问题不在于物，而在于人——是少数权力者规定了审美，是穿着者强加了价值。”这种将一切归结为“少数人权力游戏”的论调，实则是一种思想上的懈怠。它掩盖了一个更普遍的现实：在日常中，我们绝大多数人并非时尚的共谋，而是其沉默的观众与碎片化的表演者。我们并非被自上而下地灌输审美，而是主动参与了一场由资本逻辑导演的、关于身份的全景式表演。


我们在抖音、小红书等平台上所见的“奇装异服”，与其说是身份的真诚宣告，不如说是一次性的身份摆拍。正如“东北雨姐”建构的乡土符号，亦或是成都街头男同博主的夸张表演那样，服装在此不再是身份的延伸，而是流量的诱饵。直播结束，服装所承载的临时人格便即刻卸下。情境主义国际的创始人居伊·德波在《景观社会》中的批判于此显得无比前瞻：“在现代生产条件无所不在的社会里，生活本身展现为景观的庞大堆积。直接存在的一切全都转化为一个表象。”当代生活本身已经展现为景观的庞大堆积，而短视频平台上的服装表演，正是景观最集中的体现。所有真实的生活都被转化为一个表象（服装），而这个表象却比真实更加真实。


这引出了一个更为尖锐的问题：当我们嘲笑这些表演者，或凝视AV中身着制服的“老师们”时，我们在结构上享乐着何种同一性？JK勒肉的丝袜、“男同”捆缚的皮带、JK引人注目的校服、“LGBT”身穿的彩虹衣服——这些服饰在符号学上又究竟有什么样的结构性差异？


答案是窥视的权力。对前者（摆拍者）的嘲笑，是对“滑稽”符号的消费，它巩固了观看者自身“正常”的优越感；对后者（AV演员）的凝视，则是对“情色”符号的消费，满足着征服的幻想。在这两种情形中，我们享乐的对象都不是真实的人，而是一个被服装精心包裹的、安全的符号。JK制服、紧绷的皮带——这些物质本身并无区别，它们都是欲望剧场中，用以激活特定剧本的标准化道具。哲学家吉尔·德勒兹或许会将其视为“欲望机器”的装配部件，这些部件不再生产真实的欲望，而是不断再生产着资本循环所需的空洞刺激。


于是，我们抵达了现代最深的疾病：对身份认同的普遍阳痿。服装，这个曾无比强大的社会象征系统，正在失效。历史上，华服是通往社会认可的阶梯，褴褛衣衫是回归自然的宣言。而今天，一切趋于均质。精神病患的制服，旨在抹平危险的个性；昔日权贵的品级官服，已化为历史的尘土；即便是曾经的规则颠覆者如马云，最终也臣服于毫无个性的西装之下。军队的制服或许是唯一的例外，它揭示了一个残酷的真相：唯有在最极致的、前现代式的暴力组织中，服装的等级与象征意义才能被完整地保留和执行。社会学家齐格蒙特·鲍曼用“流动的现代性”来描述这个一切固着之物都趋于溶解的时代，而服装意义的蒸发，正是这种流动性最表层的症候。


在制度的空转中，我们的身体沦为执行重复指令的生物机器。而服装，这片曾经充满象征与对话的领域，也随之坍缩为一片无意义的空白。它不再言说我们是谁，仅仅标示出我们所在的功能性位置，或者，更可悲地，什么也不标示。


# 奢侈等级的虚伪——非洲的眼泪

在更深入之前，让我们来看这样的一个笑话：你去上海旅游，打算找个咖啡馆坐下喝一杯咖啡，结果主理人说自己埃塞俄比亚的咖啡豆已经预定了，只有云南豆了你还要喝吗？

这个笑话的苦涩内核在于：咖啡确实起源于非洲，但关于咖啡的文化解释权与价值定义权，早已不在非洲人手中。难道香奈儿对埃及棉花的采购，爱马仕对非洲鳄鱼皮的垄断不也恰恰反映了这一点吗？这背后是爱德华·萨义德在《东方学》中所揭示的逻辑的延续：“东方主义是一种支配、重构东方并对之行使权力的西方风格。”全球南方（曾经的“东方”）被设定为原材料与廉价劳动力的提供者，而其文化和价值，则需要由西方来“赋予”和“定义”。


非洲，特别是埃及，是世界上最重要的特长绒棉产区。埃及吉萨系列的棉花，以其极长的纤维、丝般的光泽和卓越的强度而闻名于世，是制作高端衬衫、奢华面料和高级时装的首选原料。然而，这条供应链却是一个精密的价值剥夺装置：非洲提供顶级的原材料，而欧洲则垄断了赋予其符号价值的“工艺”与“血统”。马克思在《资本论》中所分析的“商品拜物教”在这里呈现出新的形态：“商品形式的奥秘不过在于：商品形式在人们面前把人们本身劳动的社会性质反映成劳动产品本身的物的性质，反映成这些物的天然的社会属性。”我们崇拜的不仅是商品本身，更是附着于其上的、被精心建构的欧洲工艺神话，这个神话系统地遮蔽了原料的真实生命历程。


历史学者瓦尔特·罗德尼在《欧洲如何使非洲欠发达》中的分析，为理解这一现象提供了深刻的历史维度。他指出：“非洲的欠发达与欧洲的发展是同一枚硬币的两面。”非洲并非“天然”落后，而是在殖民过程中被系统性地重塑为欧洲工业的原材料附庸。今天的奢侈品供应链，正是这一历史逻辑在全球化时代的精致化延续。在埃及尼罗河三角洲，棉农在烈日下手工采摘这些“白色黄金”，他们的年收入可能不及欧洲纺织厂技师的一个月薪水，更遑论与一件香奈儿外套的售价相比。这种悬殊并非市场“自然”形成，而是几个世纪殖民掠夺所固化下来的全球劳动力分工的结果。


所以，香奈儿购买的从来不是“非洲棉花”，而是“用非洲顶级棉花在欧洲顶级工厂织造出的顶级面料”。奢侈品更倾向于营销其设计、工艺、血统和“法国/意大利制造”的终极价值。一件香奈儿外套的售价主要源于其品牌溢价，而非单纯的棉花原料。这背后是一套完整的文化霸权运作：原材料被“去非洲化”，然后被“再欧洲化”，最终以完全脱离其出身的高昂价格，卖回给全球的消费者。瓦尔特·本雅明在《机械复制时代的艺术作品》中提出的“灵晕”概念，在奢侈品这里被人工地制造出来——“即使在最完美的艺术复制品中也会缺少一种成分：艺术品的即时即地性，即它在问世地点的独一无二性。”它不是原作在机械复制时代的消逝，而是通过一套关于起源、手工和传承的叙事被人为注入的。


具体而言，意大利科莫湖地区的纺织厂通过“高支纺纱”技术将埃及棉纺成比头发丝还细的纱线，再经过“双丝光处理”使其获得永久的光泽和丝滑手感。这些复杂的工序确实创造了卓越的品质，但更重要的是，它们构建了一个无法被简单复制的“工艺神话”，这个神话成为了品牌溢价的核心支撑点。人类学家阿尔君·阿帕杜莱在《物的社会生命》中提出的“全球文化流动”理论在此极具解释力：“商品，如同人一样，具有社会生命。”埃及棉花作为“商品流”的旅程，必须与欧洲的“技术流”、品牌的“媒体流”和消费主义的“意识形态流”交汇，才能完成其从原材料到奢侈品的蜕变，而在这个过程中，非洲的主体性被系统地边缘化了。


即便你可能会说，奢侈品公司在当地开设工厂，帮助当地人脱困，为其代言，不也是在做善事吗？但这种慈善姿态，恰恰是齐泽克所批判的文化资本主义的典范。它让你在消费的同时，不必感到良心不安。


哲学家齐泽克在《变态者意识形态指南》中分析星巴克时一针见血地指出：“你多付了点儿钱，你就不仅仅是个消费者了，你还贡献了一份责任。对环境、对穷人、对非洲饥民等等的责任，我认为这就是消费主义的终极形态。”这种将伦理消费嵌入商品逻辑的做法，完美地化解了潜在的批判，使得系统性的剥削被转化为个人可以通过购物来赎买的“罪责”。


然而，最令人忍俊不禁的悖论在于：我们在消费咖啡时可能还会被提醒存在所谓的人道主义，但我们在消费奢侈品时，却几乎完全不会意识到其原料生产国的面貌。这种对原料来源的系统性忽视，使得奢侈品的光环建立在对全球南方劳动与资源的象征性抹除之上。正如同美国人在购买香蕉时永远不会想到“香蕉共和国”的血泪史，消费者抚摸着爱马仕鳄鱼皮包时，脑海里浮现的也只是巴黎的浪漫，而非非洲河流的泥泞。人类学家阿尔君·阿帕杜莱所言的全球“物之社会生命”在此被斩断，奢侈品作为“物”的传记，其关键的前半生被刻意地遗忘和擦除。


# 身份的真空：从“服装即身份”到“感觉不如原神”

在考察了一个简短的服装史和其生产过程后，我们可以很简单地得出一个结论——“服装即身份”的古老信条已然崩塌。服装在今天，至多只能模糊地暗示我们的职业范畴，但无法定义我们是谁。我们正生活在社会学家欧文·戈夫曼在《日常生活中的自我呈现》中所描述的“角色扮演”日常中：“当个体在他人面前呈现自己时，他的表演往往会趋向于体现和例证社会正式认可的价值观，而实际上他的全部行为并不具备这种价值。”但可悲的是，我们连一个稳定的角色剧本都已丧失，只剩下临时的、可随时替换的角色碎片。

这一过程有其历史根源。工业革命的到来催生了“成衣”的大规模生产，开始系统地抹平服装在外观上的阶级差异。当资产阶级的绅士服和工人的工作服在流水线上被同样地制造出来时，服装作为阶级标识的功能就已经开始松动。而晚期资本主义的到来，则加速了这一进程，使其达到顶峰。


让我们来看看这两个看似不相关的案例：南京红姐身穿黑色丝袜，穿着充满诱惑力的上衣，用口罩覆盖住了男性特征，反而让她的一种经过精心计算的“女性气质”更为突出，创下了千人斩的纪录；普京西装革履，手戴名表，在社交媒体上公开宣称浏览色情网站相当正常。


这两个案例共同指向了同一个真相：服饰已经无法稳定地代表任何内在本质或真实身份。普京的西装不再代表庄重的承诺，反而成为其表演“率真”的道具；红姐的性感装扮则彻底沦为流量狩猎的标准化装备。朱迪斯·巴特勒在《性别麻烦》中提出的“性别操演论”可以扩展到整个身份领域：“性别不应该被解释为一个稳定的身份或是产生各种行为的能动性的场域；相反，性别是在时间中建立的一种脆弱的身份，通过风格/程式化的重复行动在一个表面的空间里建构而成。”身份不再是内在的表达，而是通过重复的符号行为（如穿衣）操演出来的。但当这种操演变得过于随意和工具化，其本身也就失去了可信度。正如前文所言，作为遮羞最原初的工具已经失效，我们肆无忌惮地玩着“露出”的游戏，政客则肆无忌惮地发表着黄色言论。


故此，我们所缺少的恰是对羞耻的反思，并且必须警惕我们在羞耻中的隐秘享乐。德国社会学家诺贝特·埃利亚斯在《文明的进程》中详细追溯了从中世纪到近代，欧洲社会如何通过一系列关于身体行为、礼仪和羞耻感的规范来建构“文明化”的主体。而当代的“无羞耻”文化，似乎标志着这一漫长文明化进程的某种逆转或深刻变异。我们在对羞耻的漠视中，并未走向真正的解放，反而走向了性冷淡与情感阳痿的极端。这一事实在晚期资本主义中也得到了证实，在这种现实中，完全的性放任并未带来狂喜，反而导致了焦虑、ED和情感的枯竭。正如拉康所警示的，当代社会的命“去享乐！”本身就成了最严酷的超我，它压垮了欲望，而非释放了欲望。哲学家韩炳哲在《倦怠社会》中进一步阐述：“21世纪的社会不再是一个规训社会，而是功绩社会。它的成员不再是‘服从主体’，而是‘功绩主体’。他们是自身的雇主。”这种“自由的暴政”导致了一种自我剥削，最终使得主体精疲力尽，情感淡漠。


服饰象征的失效，也必然产生阶级印象的真空。在以往，衣衫褴褛的无产者一般被认为是肮脏、素质低下的；遍身罗绮的资产阶级则被认为是高雅、上流的。今天，科技新贵们穿着连帽衫就能掌控万亿资本，而精心装扮的都市白领可能只是系统内一颗可随时替换的螺丝钉。皮埃尔·布尔迪厄在《区隔》中所分析的：“品味分类并分类着分类者。”通过品味和生活方式进行的阶级区隔虽然仍在运作，但其符号变得空前混乱和不可靠。当马克·扎克伯格穿着灰色T恤出现在亿万观众面前时，他并非在展示“无风格”，而是在展示一种新的权力姿态——我无需用传统象征成功的服装来证明自己。服装，这个曾经最直观的社会阶层说明书，已经彻底破产。我们进入了一个身份流动、符号错乱、一切固着的意义都在蒸发的新世纪荒原。德勒兹在《关于控制社会的后记》中预言的“控制社会”：“控制社会不再通过禁锢运作，而是通过持续的控制和即时的信息传播。”它不再主要通过禁锢身体，而是通过引导和调制我们的欲望、消费和身份认同来实现，而服装，正是这场流动控制中最浮于表面的浪花。


在这个意义上，当我们说“感觉不如原神”时，我们戏谑地承认：在虚拟世界中一个精心设计的头像、一套限定皮肤，所能带来的身份认同感与社群归属感，远比现实世界中我们身着的任何一件服装都更加强烈和确定。这引向了我们对未来的最后展望：虚拟时尚的兴起。在元宇宙中，服装彻底摆脱了物理世界的最后束缚——它不再需要保暖、保护甚至遮羞，它成为了纯粹的信息、纯粹的符号。这既是服装身份功能的终极解放，也是资本开辟的新战场。奢侈品巨头们早已布局数字服装，售卖仅存在于屏幕上的虚拟手袋和衣物。而这或许是这个时代最悲哀的注脚：服装，这个人类最古老的符号发明，最终在虚拟世界的降维打击下，显露出了它在现实世界中意义的空空如也。我们仿佛集体患上了韩炳哲在《非物》中所诊断的“否定性的消失”病症：“世界变得越来越贫穷，越来越缺乏经验，因为它被剥夺了否定性。一切都被暴露在一种毫无抵抗的可视性中。”在一个充满过度曝光、过度连接和过度符号的世界里，真实的、有深度的身份认同（这需要他者和边界的否定性）已无处安放，只能遁入虚拟的、可随意切换的皮肤之下。服装，这个曾经的现实身份铠甲，在数字灵魂的迁徙中，沦为了被遗弃在岸边的空壳。
